# Claude Code Prompt: IL Shield Landing Page

## Context

You are building the landing page for IL Shield Protocol — a parametric impermanent loss protection protocol for Uniswap v4 LPs. The site is deployed at il-shield-protocol.vercel.app (Next.js on Vercel).

The current page is basically empty — just a title. We need a full, polished landing page that serves two audiences: **grant program reviewers** (who need technical credibility) and **future users** (who need simplicity).

## What IL Shield Does (Product Context)

IL Shield is insurance for Uniswap LPs. LPs pay a streaming USDC premium and receive payouts when impermanent loss exceeds their fee income.

Key facts:
- Parametric insurance: streaming USDC premiums → payouts when IL > fee income
- Tranched underwriting: Senior vault (8-12% APY, last-loss) + Junior vault (20-50% APY, first-loss)
- No native token. Stablecoin collateral only. Non-transferable ERC-721 protection NFTs.
- Architecture: ILShieldCore, PricingOracle, ILPNRegistry, SeniorVault, JuniorVault, ILShieldHook (optional v4 hook)
- 216 tests passing, deployed on Ethereum Sepolia + Unichain Sepolia
- Previous attempts failed: Bancor (minted tokens to pay claims → collapsed), Panoptic (options-based, complex)
- IL costs Uniswap LPs $1.5-3B/year. 60% of non-stablecoin LP positions end up unprofitable.

## Requirements

### Tech Stack
- **Next.js** (already set up on Vercel)
- **Tailwind CSS** for styling
- **Framer Motion** for subtle animations (scroll-triggered reveals, not flashy)
- **Dark mode only** — dark navy/charcoal background, DeFi standard
- No external UI component libraries needed — keep it lean
- Fully responsive (mobile-first)

### Design Direction
- Clean, minimal, professional. Think: Morpho, Eigenlayer, or Aave's landing pages
- NOT "crypto bro" or meme-coin energy
- Color palette: dark navy (#0a0e1a or similar) background, electric blue (#3b82f6) or shield-gold (#f59e0b) accent, white text
- Typography: Inter or similar clean sans-serif
- Generous whitespace. Let the content breathe.
- Stats should be prominent and large
